# Examples

The `examples/` folder contains basic projects initialized with the PowerSync CLI. Each example demonstrates a different setup and can be used as a starting point or reference.

## Examples

| Example                                                                            | Description                                                                |
| ---------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| [**Basic Cloud Pull**](./cloud/basic-cloud-pull/README.md)                         | Cloud instance config pulled with `powersync pull config`.                 |
| [**Self-Hosted: Basic Supabase**](./self-hosted/local-basic-supabase/README.md)    | PowerSync with external DB and storage (Supabase).                         |
| [**Self-Hosted: Basic Postgres**](./self-hosted/local-postgres/README.md)          | PowerSync with Postgres database and storage in Docker.                    |
| [**Self-Hosted: Postgres + Node.js**](./self-hosted/local-postgres-node/README.md) | Postgres stack with Node.js app using PowerSync SDK and shared HS256 auth. |
